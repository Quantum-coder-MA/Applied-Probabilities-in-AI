# Applied Probabilities in AI

This repository contains a university project showcasing applications of probability theory and AI through an interactive Streamlit dashboard.

## Overview

The project includes:
- `app.py`: a Streamlit app demonstrating probability distributions, sports injury modeling, and breast cancer analysis.
- `multimodal_sports_injury_dataset.csv`: sports injury dataset used in the app.
- `dataCancer.csv`: breast cancer dataset used for analysis.
- Several Jupyter notebooks exploring probabilistic models and clustering.

## Features

- Simulation and visualization of probability distributions
- Exploratory data analysis for sports injuries
- Breast cancer data analysis with statistical insights
- Custom dark academic UI styling for Streamlit

## Getting Started

### Requirements

- Python 3.8+
- `streamlit`
- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`
- `scipy`

### Install dependencies

```bash
pip install streamlit pandas numpy matplotlib seaborn scipy
```

### Run the app

```bash
streamlit run app.py
```

Then open the local Streamlit URL shown in the terminal.

## Repository Structure

- `app.py` — main Streamlit application
- `dataCancer.csv` — breast cancer dataset
- `multimodal_sports_injury_dataset.csv` — sports injury dataset
- `cancer_bayesKNN (2).ipynb` — notebook exploring Bayesian and KNN methods
- `cancer_clustering_metriques (1).ipynb` — notebook on cancer clustering metrics
- `probabilistic_hybridation_cancer.ipynb` — probabilistic hybridization analysis
- `sports_injury.ipynb` — sports injury data exploration
